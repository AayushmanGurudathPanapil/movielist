🎬 Movie App

A React-based movie browsing application that allows users to explore movies and manage a personalized favorites list. The app uses the React Context API for global state management and modern React hooks for a clean and scalable architecture.

🚀 Features

1. Browse and display movie cards

2. Add and remove movies from Favorites

3. Global state management using Context API

4. Reusable components and modular structure

5. Responsive and dynamic UI updates

🛠️ Tech Stack

1. React
2. Vite
3. JavaScript (ES6+)
4. CSS
5. API integration
